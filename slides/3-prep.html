Authentication

Who are you?

Authentication is the process or action of verifying the identity of a user or process.

BasicAuth

n the context of an HTTP transaction, basic access authentication is a method for an HTTP user agent (e.g. a web browser) to provide a username and password when making a request.

Token Based Authentication

Token-based authentication is a protocol which allows users to verify their identity, and in return receive a unique access token. The user retains access as long as the token remains valid. Once the user logs out or quits an app, the token is invalidated.

Authorization

Authorization is the function of specifying access rights/privileges to resources (human or machine).
